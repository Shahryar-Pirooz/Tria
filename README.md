# 🔐 Tria Password Manager

Welcome to Tria! 🌟

Tria is a friendly little password generator that creates a password from three simple pieces of information: your name, a memorable code, and the website or app domain. Enter the same values again later, and Tria will recreate the same password for you. ✨

## 🧠 How it works

Just provide:

- a name (for example, Michael)
- a simple code (for example, 1234)
- a domain (for example, facebook.com)

Tria combines those inputs into a password you can reuse whenever you need it. No complicated setup, no stress. 😊

## ✨ Why people like Tria

- simple and easy to use
- deterministic password generation
- no need to memorize a long password
- works right in your browser
- perfect for quick, repeatable passwords

## 🚀 How to use it

1. Enter your name.
2. Enter your memorable code.
3. Enter the domain.
4. Generate your password.

You can come back anytime with the same details and get the same result. 💙
